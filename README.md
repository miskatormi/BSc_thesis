## Abstract 
*Note: This abstract was originally written in Finnish and for this Readme translated into English. (The original abstract can be found within the PDF file)*

The study of options is essential, as options enable financial risk mitigation and facilitate a deeper understanding of financial markets. This thesis explores the fundamental nature of options and examines how they are modeled mathematically. Additionally, a computational example of a specific option is presented, alongside an analysis of how the application of the model changes when parameter estimation errors are introduced.

The mathematical framework utilized in this thesis is the binomial asset pricing model, which can be traced back to William Sharpe’s presentation in 1978. The model was further developed for option pricing by Cox, Ross, and Rubinstein in 1979. It is fundamentally grounded in the probability theory of finite sample spaces. A measure-theoretic and probabilistic exposition of the model was presented by Shreve in 2004, which serves as the primary source for this thesis. This thesis derives the binomial model from probability theory and provides proofs for the core results concerning options within the binomial framework.

Furthermore, this thesis establishes a formal definition for options and proves a central theorem essential to their analysis. This theorem pertains directly to option pricing and its application in risk management and hedging. The theorem is subsequently applied to a computational example that incorporates the possibility of estimation error. The thesis concludes that while estimation error reduces the accuracy of the binomial model, it does not render it obsolete. However, the estimation error model implemented in this study does not fully capture the practical deployment challenges faced by the binomial model, nor does it encompass all potential types of errors.
